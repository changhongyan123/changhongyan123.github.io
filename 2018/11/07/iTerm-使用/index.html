Terminal工具集合

为什么使用iterm2？

苹果自带的terminal确实效果还是可以的，并且功能其实是齐全的。但是当我接触过了iterm2 之后真的爱上了这个更加灵活的terminal了。个人觉得有几个优点

  • 方便的快捷键。
    分分钟打开新的页面，左右分屏，上下分屏。
    新建窗口: command + t
    关闭窗口: command + w
    前一个窗口: command + <- 后一个窗口:="" command="" +="" -="">
    进入窗口123: command + 1,2,3
    横向新建窗口: command + d
    竖向新建窗口: command + shift +d
    垂直分割: command + d
    水平分割: shift + command + d
    前一个面板: command + [
    后一个面板: command + ]
    切换到上/下/左/右面板: Option + command + 上下左右方向键

  • 自定义界面
    我使用的是oh-my-zsh的扩展,加上自定义皮肤之后感觉瞬间舒服了很多。
    链接https://github.com/robbyrussell/oh-my-zsh

常用命令

  • top 查看后台程序
  • jobs
  • ssh -i 指定ssh key的位置去进行链接
