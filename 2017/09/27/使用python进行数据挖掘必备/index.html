使用python进行数据挖掘必备

之前一直使用python做一些数据挖掘和机器学习相关的项目，现在整理一下必备的module和安装过程

python install

由于kera在Windows只支持python3.0+所以在python安装时候选择3.0+版本。python3.4+版本已经在下载的时候安装了pip组件，所以可以直接使用pip install安装python module

  • python 下载地址
    安装numpy
    pip install numpy
    安装scipy
    因为scipy依赖于其他组件，直接使用pip install scipy极有可能不成功，所以下载whl文件。
    pip install scipy-1.0.0b1-cp36-cp36m-win32.whl
    (32bit python:3.6 scipy:1.0)
    下载地址
    安装jupyter
    pip install jupyter
    在想要建立笔记的文件夹下面运行
    jupyter notebook
    就可以开始使用jupyter，一种笔记类型的组件，运行效果可以直接看到
    More
    安装pandas
    pip install pandas
    文件处理module，以类似sql方式处理csv，excel等文件形式。
    More
    安装matplotlib
    pip install matplotlib
    matplotlib是一个画图module，多种呈现方式。还有一个seaborn是基于matplotlib的另外一个module，配色更加好看。
    More
    seaborn
    安装机器学习模块：sklearn
    pip install -U scikit-learn
    其中包括了很多机器学习常用的模型算法，直接调用api可以使用，非常方便。
    Learn Sklearn
    安装深度学习模块：keras
    keras是基于tensorfolw和theano等神经网络API，非常好上手，适合新手。由于我的电脑GPU使用不起，所以这些安装过程是按照cpu的方式安装。如果想尝试gpu可以点击下面链接找相关的方式。由于我下载的python版本是32bit不能安装tensorfolw导致keras有问题，所以不在下面演示。
    Learn Keras
