实习经历

现在是我在S公司实习的第九个月，九个月的时光是充满收获和感动的。现在总结一下我在实习期间的一些经历和感悟。

找工作

作为我们专业的学生，在这个方面比起其他人有着先天的优势。我们专业培养方案要求大三下学期到公司实习半年，有很多合作的企业会来学校进行招，S就是合作公司中的一个。除此之外，也可以关注各个招聘网站，里面会发布一些招聘信息。另外还有一条捷径是内推了，这时候就要靠你优秀的学长学姐的啦～
关于工作的选择：对于技术能力不是很强的学生，并且之后想参加工作的，建议到初创型公司接受锻炼。根据同学反馈，初创型公司的工作量比较大，工作员工比较缺少。在这样的环境中进行锻炼，可以学到很多技术方面的知识，锻炼代码能力。

面试

我总共参加了三场实习面试，因为我之前做的是自然语言处理方面，所以对python比较熟悉，java也是之前开发APP的基础。第一家公司的笔试个人没怎么写，里面是考java的。第二个公司就是S，因为是外企的原因比较注重英语能力，所以有一个考核是根据一个英语描述的场景以英语的方式说出解决方案，之后就是群面，讲项目经历。第三个面试是电面，因为是内推，所以个人感觉不太难，主要是问一下建立上面的问题。
不同的公司对于人才的需求是不一样的，不过大部分的公司都比较注重你的学习能力和态度，其实真正掌握多少技术个人感觉没那么重要。

总结

选择一个适合的公司是极其重要的，合适并不代表一定是大公司或者是名企，关键在于你想从实习中获得什么。

比起技术公司可能更看重态度 和学习能力.
