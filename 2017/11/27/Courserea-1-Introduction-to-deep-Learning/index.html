Courserea(1) Introduction to deep Learning

Neural Network and Deep Learning(week 1)

Note:

  • Deep Learning: 一般指的是训练神经网络，有时候是比较大的神经网络。
  • Supervised Learning:是机器学习的一种方法，也是深度学习多利用监督学习来创造价值。所谓监督学习，类似于给一个小朋友10000张有猫咪的图片，10000张狗狗的照片，并告诉他这是猫咪还是狗狗，来让小朋友学习。监督学习和无监督学习最大的差别可能是训练数据是否已经知道label（狗狗还是猫咪）。
  • 结构化数据：类似于数据库数据，有各个column的定义。如同预测房价中考虑房子的面积，房子的地理位置。
  • 非结构化数据：类似音频，图像，文字等。unstructured data 更加不好处理，难度更大。
  • 数据量和模型之间的关系：当数据量不达到一定数量的时候，模型之间的差距并不能体现，当数据量达到一定规模之后才会呈现模型的优良。[NG课程里面有一个曲线图，很好的说明该问题 :why is Deep Learning taking off?]

这周课程比较基础，适合于初学者。更多的重点在解释监督学习以及Neural Network等基础的内容上面，对于深度学习解释透彻。
