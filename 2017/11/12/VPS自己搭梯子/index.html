VPS自己搭梯子

VPS自己搭梯子

由于学习（！！！）的需求，不知道在VPS上面做过多少手脚了，总结一下防止下次搭梯子又要baidu一下～

购买VPS

我是使用的github的学生认证，送$50的digital ocean的服务器券。
使用前会绑定paypal信息，并支付$5作为初始资金，这样算下来有$55。
网上有一些邀请码，会有$10的优惠券，如果先绑定这个优惠券那么学生优惠券就会无效，需要跟跟客服小姐姐联系。一般在给你加$40，毕竟优惠不能共享，有时候遇到nice的客服，就会跟你全部加上，那么也就是$65！！！（我就是这样的）
费用：只用VPN的话，$5一个月足够使用，那么也就是可以使用将近一年呢～
之后选择data center，创建一个就ok了～用户名和密码会以邮件的形式发给你哦～

安装ss

利用ssh连接到服务器之后，就可以干事搭梯子了～
连接可以使用putty，也可以使用mobaXterm！！（良心推荐）
之后按以下步骤进行：

更新软件源

sudo apt-get update

安装pip

sudo apt-get install python-pip

安装shadowsocks

sudo pip install shadowsocks

创建/etc/shadowsocks.json文件

创建完毕后，赋予文件权限：

sudo chmod 755 /etc/shadowsocks.json

安装支持加密模块

sudo apt–get install python–m2crypto

后台运行

sudo ssserver -c /etc/shadowsocks.json -d start

设置开机自启动
编辑 /etc/rc.local 文件

sudo vi /etc/rc.local
在 exit 0 上边加入如下
/usr/local/bin/ssserver –c /etc/shadowsocks.json

在android或者电脑端安装shadowsocks的客户端，输入相应的ip，密码，加密方式就可以使用了！
电脑客户端软件

设置ipsec

针对于ios用户，可以在VPS设置ipsec。
一键安装：

wget https://git.io/vpnsetup -O vpnsetup.sh && sudo sh vpnsetup.sh

安装后会自动显示用户名，密码，密钥
在ios vpn设置中输入相应内容即可～
